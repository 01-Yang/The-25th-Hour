## 状态收益修正

重复选择同一行动不做固定衰减，收益由精力和压力决定。

| 状态 | 条件 | 影响 |
|---|---|---|
| 状态良好 | 精力 >= 60 且压力 <= 50 | 正常收益 |
| 状态波动 | 精力 35-59 或压力 51-75 | 正向收益 -10% |
| 状态危险 | 精力 20-34 或压力 76-85 | 正向收益 -20%，给提醒 |
| 崩溃边缘 | 精力 < 20 或压力 > 85 | 正向收益 -35%，事件风险提高 |

提醒只保留三类：

- 精力见底
- 压力过载
- 状态危险

## 个人属性加成

个人属性既会被行动增长，也会反过来影响行动结果。

| 属性值 | 加成 |
|---|---|
| 0-39 | 对应收益 -15% |
| 40-59 | 正常 |
| 60-79 | 对应收益 +10% |
| 80-100 | 对应收益 +20% |

## 评图

```text
评图总分 = 进度 40% + 作品质量 35% + 汇报策略结果 25% + 导师任务修正
```

等级与绩点：

- A 哥们你封神了！：`4.0`，总分 `>= 85`
- B 做的不错！：`3.3`，总分 `75-84`
- C 平庸之作。：`2.7`，总分 `60-74`
- D 你，过关！：`2.0`，总分 `45-59`
- F 你挂科了！：`0.0`，总分 `< 45`

GPA 结算规则：

- 开局显示 `GPA = 4.0`
- 每学期先根据评图等级得到“设计课绩点”
- 再叠加“大学课程修正”
- 得到该学期最终绩点后，再并入个人累计 GPA

大学课程修正：

- 优：`+0.2`
- 稳：`+0.0`
- 危：`-0.2`
- 崩：`-0.5`

累计规则：

```text
个人GPA = 所有已完成学期最终绩点的平均值
```

## 模型周

模型周只影响作品质量。

材料选择：

- 便宜材料：作品质量 +1，花费低，模型事件风险高
- 标准材料：作品质量 +3，花费中等，风险中等
- 高级材料：作品质量 +5，花费高，风险低

模型周事件修正：作品质量 -3 到 +4。

## 保研资格

基础保研资格：

- GPA >= 3.6
- 最近 4 学期评图没有 F
- 最近 4 学期至少 2 次 B 或以上
- 作品集 >= 80
- 至少 1 次竞赛获奖，三等奖及以上

保研档位：

- 顶级档：GPA >= 3.85，作品集 >= 90，至少 1 次二等奖及以上
- 老八校 / 985 档：GPA >= 3.65，作品集 >= 82，至少 1 次三等奖及以上
- 普通一本升学档：GPA >= 3.3，作品集 >= 72，无 F

## 竞赛
