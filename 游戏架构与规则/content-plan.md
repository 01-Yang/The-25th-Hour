## 角色被动特质

| 角色 | 被动特质 | 核心感觉 |
|---|---|---|
| 普通人 | 稳稳地来 | 不冒头，但不容易失控 |
| 混的入 | 低空飞行 | 可以烂着过，但很难封神 |
| 不吃压力之人 | 钝感力 | 压力来了也能先睡一觉 |
| 设计赋能哥 | 会讲故事的人 | 同样的东西，他更会讲 |
| 寒门贵子 | 不敢停下 | 一停下来就会慌 |
| 吃满压力之人 | 越压越狠 | 越到极限越会咬牙硬顶 |
| 未来的老板 | 资源调度 | 更懂得把钱变成效率 |
| 投胎专家 | 松弛预算 | 资源宽裕，但花钱也快 |
| 基因叛逆者 | 逆反驱动 | 越被否定越想证明自己 |
| 小镇做题家 | 应试惯性 | 在学业规则里最会活 |
| 柯布西耶继承者 | 圈内惯性 | 很早就知道门往哪边开 |

## 导师阶段任务

| 导师 | 阶段任务 | 完成奖励 | 未完成处罚 | 事件方向 |
|---|---|---|---|---|
| 王老师 | 接设计外包 >= 2 或实习 >= 1 | 作品质量 +4，设计水平 +2 | 评图总分 -3 | 场地、实习、城市观察 |
| 戈老师 | 方案推敲 >= 4 且阅读/展览/讲座 >= 3 | 作品质量 +3，GPA学期修正 +0.10 | 评图总分 -2 | 讲座、图书馆、学术 |
| 林老师 | 竞赛投稿 >= 1 或作品质量 >= 75 | 作品质量 +5 | 压力 +8，评图总分 -3 | 返工、竞赛、高压修改 |
| 陈老师 | 场地调研 >= 3 且阅读/展览/讲座 >= 3 | 作品质量 +4，创意审美 +2 | 评图总分 -2 | 雨天、夜谈、理想讨论 |
| 周老师 | 学习AI和设计软件 >= 5 且软件技术 >= 70 | 软件技术 +2，作品质量 +3 | 评图总分 -2 | 插件、软件、AI争论 |
| 许老师 | 进度 >= 70 且本学期没有 F | 压力 -6，抗压能力 +2 | 评图总分 -1 | 生活、自主探索、恢复 |
| 韩老师 | 竞赛投稿 >= 1 或名企实习 >= 1 或本学期 A | 设计水平 +3，评图总分 +4 | 压力 +10，评图总分 -3 | 名企、顶级竞赛、极限熬夜 |

- 阿尔瓦罗·西扎：莱萨温泉
- 扎哈：广州大剧院
- SANAA：金泽21世纪美术馆
- 王澍：宁波博物馆
- 隈研吾：浅草文化观光中心

## 事件内容原则

- 事件标题要短。
- 事件文本要有建院生活感。
- 不直接照搬外部游戏文本。
- 随机事件不提供选择。
- 交互事件提供选择。
- 事件结果必须落在既有体系内。
