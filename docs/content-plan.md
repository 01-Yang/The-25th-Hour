## 角色简介

| 角色 | 简介 |
|---|---|
| 普通人 | 你是建院里最不起眼的那一个，就像专教里那把没人抢的椅子，一直在那儿，不声不响。 |
| 混得入 | 你高考掉档来了双非，但你是真的不在乎。交图周别人通宵，你也在通宵——打游戏。 |
| 不吃压力之人 | 你是那种评图被批得再惨，你也能笑着跟老师说“那我回去改改”的人，你说人生除了生死，其他都是擦伤。 |
| 设计赋能哥 | 你在设计上有开挂的天赋，同样一个题目，别人还在想怎么排功能，你已经随手画出了三版空间方案。 |
| 寒门贵子 | 你是村里第一个考上名校的，你妈在你出发前为你包了顿饺子，你爸没说什么，只是把学费塞进你书包最里层。 |
| 吃满压力之人 | 你是大专生，家里穷得连你上学的路费都是找邻居凑的。你不爱提这些，只是沉默地打工、上课、赶图。 |
| 未来的老板 | 你家是开建材公司的，而你似乎也是那种天生会做生意的人，你对“钱”的嗅觉比设计感还灵敏。 |
| 投胎专家 | 你的高考分数只能上专科，但你爸说了“建筑这行看人脉”。你从来不为钱发愁，家里有矿的你开着车来上学。 |
| 基因叛逆者 | 你是院士家的“废柴”，高考连二本线都没够着。你对自己说：当不了接班人，就当个普通人吧。 |
| 小镇做题家 | 你是县城一中出来的，高考排名全校第一。你擅长把任何知识点变成笔记、思维导图、题库。 |
| 柯布西耶继承者 | 你从小跟着父母出入各种学术会议，见过的建筑大师比同学认识的明星还多。 |

## 评图结果弹窗文案

| 等级 | 弹窗文字 |
|---|---|
| S | 从城市尺度到人体模数的推敲贯穿始终，体量生成与环境互为因果，平面遵循“自由平面”的开放逻辑，剖面则运用“建筑漫步”的叙事，空间序列随光线移动呈现节奏，构造与结构逻辑高度统一，材料表达诚实。该方案已达到专业实践层面的完成度，是建筑学基础训练的优秀范例。 |
| A | 图纸从场地分析到体量生成的逻辑链条完整且紧凑。空间序列的起承转合被你处理得很克制，入口的收、中段的放、高潮处的留白，每一处都经过推敲。剖面表达出了丰富的竖向层次，光影成为空间叙事的一部分。但缺点同样明显，构造节点深度不足，效果图中的光影氛围虽然动人，却与剖面上的开窗位置存在矛盾。 |
| B | 整体方案可圈可点，形体空间逻辑成立，体量生成有依据，但“路径与节点”的节奏控制还不够老练。你在入口处设置了一个很棒的过渡空间，但之后的序列突然变平，缺少起伏。剖面中有一段高度变化处理得有些犹豫，导致竖向体验的中断。 |
| C | 图纸完成了任务书的基本要求，但缺少对“空间氛围”的主动塑造。你的体量生成过程被简化成一张“推导图”，老师读不出你的设计出发点——是回应地形？是受某种空间类型启发？还是纯粹的形式操作？“功能合理，但场所感弱，建议强化设计概念。”你盯着那行字，意识到自己这学期一直在“完成作业”，而不是“做设计”。 |
| D | 图纸完全没有体现“场所精神”。你的方案像是可以放在任何一块地上的通用盒子，与基地没有对话——不关心风向、不回应日照、不尊重地形。剖面诗学完全缺失，所有空间都是同一高度、同一进深，体验被扁平化到像一个仓库。老师写了四个字：“建筑何在？”你不知道怎么回答。 |
| F：进度未完成 | “因未完成规定图纸数量，成绩为不及格。”赶图周开始时，你的进度就落后了一截。你总觉得“还来得及”，于是概念拖了三天，模型拖了两天，排版拖到最后一天。当你发现时间不够时，已经连平立剖都凑不齐了。老师问：“你的总图呢？”你答不上来。评语栏写进度原因挂科是最真实的教训。 |
| F：质量不过关 | 平立剖之间矛盾重重——平面上的窗在立面上没有出现，剖面的高度与平面对不上。方案缺乏基本的空间逻辑——功能与形式之间没有找到平衡，体量生成像是随机堆砌。分析图与设计方案之间没有对应关系，老师无法判断你的设计意图。评语只有一句话：“未达到建筑学基本训练要求。”你不服气，但你翻开自己的图纸，发现他说的是事实。 |

## 考公岗位池 V1

| 岗位层级 | 岗位 | 简介 |
|---|---|---|
| 国家部委层 | 住建部、自然资源部 | 上岸天花板，笔试面试都得往死里卷。进去了不画图，专写红头文件，偶尔批批别人画的总图。 |
| 省市厅局层 | 文旅厅、城管局 | 省城或市里的单位，比部委好考一点。城管局可能需要你审批占道施工图，也算专业对口——毕竟你学过规划。 |
| 基层公务员 | 乡镇街道普通科员 | 乡镇街道的普通科员，离图纸很远，离百姓很近。你的建筑学在这里最有用的是看懂违建和画明白村道。你画的不是总图，是村民家门口的那条水泥路。 |

文案口径：`基层公务员 / 基础服务——为人民服务` 按低位但有尊严的普通成功维护，不写成失败惩罚或苦情回落。

## 建筑相关工作池 V1

实习的工作池读取建筑相关工作池，不包括转行岗位池。

| 岗位 | 简介 |
|---|---|
| 地方设计院 | 在家乡省份的画图工，项目不酷但能落地。工资够活，加班常态，优点是爸妈觉得你“进国企了”。 |
| 国企设计院 | 天花板级别的上岸，笔试面试都得卷。福利好，活儿也重，画不完的施工图和改不完的节点。你问为什么叫“国企”？因为连加班都是体制内的味道。 |
| 独立小型工作室 | 老板可能就是你的学长，项目小而杂，住宅、民宿、小商店都接。累了没人管，赚了大家一起分——当然大部分时间是不赚的。 |
| 外企事务所 | 工位在CBD写字楼的三十层，落地窗外是整座城市的天际线。图纸上标注从毫米换成英寸，你每天用三种语言的邮件沟通方案，时差让你习惯了凌晨两点和欧洲连线。 |
| 大师建筑事务所 | 工位隔壁可能是某国留学生，项目动不动就上杂志。加班到凌晨是常态，但拿到的工资条和你的黑眼圈成正比。值不值？看你想不想在方案落款留下自己名字。 |

## 转行岗位池 V1

| 岗位 | 简介 |
|---|---|
| AI产品经理 | 把建筑学的逻辑思维带到互联网，整天跟程序员吵架：“这功能不是我要的。”工资翻倍，但你再也没机会画剖面了。 |
| 游戏场景建模师 | 在虚拟世界里搭山盖海，不用考虑承重和排水。甲方从“您再改改”变成“这棵树穿模了”。加班依旧是老味道，但电脑配置终于能跑满帧了。 |
| 销售/商务 | 电话不停、饭局不断、微信好友全是甲方。你学会了在酒桌上谈合同，在厕所里改报价。提成到账的那一刻，你觉得被挂几次电话也值了。 |
| 新媒体内容 | 每天追热点、编标题、写“震惊体”。阅读量高了老板夸你，低了粉丝骂你。你以为逃离了改图，结果在排版和找封面图里循环往复。 |
| 插画师 | 丢掉尺规，拿起数位板，画城市速写、建筑漫画、甚至表情包。最受欢迎的作品是“通宵赶图猫猫头”。有人问“你学建筑的怎么来画画”，你说“因为画图不用算梁高”。 |
| 创业 | 最困难之路，最孤独之路，也是风险与机遇并存之路。你像是在深夜拉一条没有端点的轴线，没有图纸，没有老师，没有人告诉你下一根线画在哪，你的对手只有一个：你自己。 |
