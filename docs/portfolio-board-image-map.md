# 作品集展板图片映射稿

本文维护 `专业设计作业图片/` 目录中的评图后展板图与作品集系统课题卡的对应关系。它们只用于作品集系统，不进入简历顶部摘要、结尾回忆走马灯或结局图片映射。

## 挂载规则

- 展板图只挂在 `作品集系统 > 课程设计成果 > 单项目详情` 视图；列表卡面是否显示缩略图可由前端按性能决定。
- 只有最终图纸评级为 `B / A / S` 的课题才进入作品集并展示展板图；`C / D / F` 评级课题不生成作品集项目详情。
- 前端优先按 `semesterIndex` 读取；若展示层已经只保留课题标题，则用课题标题做二次校验，不用文件名反推学期。
- 同一学期只挂 `1` 张展板图；当前覆盖 `大一上` 到 `大四下` 共 `8` 张。
- `大五毕业设计` 当前没有新增展板图，本轮先不挂载。

## 接入提醒

- 如果 `ReviewRecord`、课程成果卡和作品集详情页同时存在，`semesterIndex` 作为主键，课题标题作为展示文本。
- 展板图资源路径建议直接写入作品集系统的课题详情配置，不要在渲染阶段按中文文件名拼路径。
- 若后续补齐 `大五毕业设计` 展板图，优先沿用同一映射表继续追加 `semesterIndex = 9 / 10` 的约定，而不是新开一套规则。
