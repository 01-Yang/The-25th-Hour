## 行动联动修正

### 画图与赶图

- `正常速度画图` 与 `爆肝通宵赶图` 只直接负责 `进度` 推进，不直接增加 `作品质量`。
- 两项行动的最终 `进度` 收益与 `精力 / 压力` 代价读取 `软件技术` 档位修正。

| 软件技术 | 正常速度画图修正 | 爆肝通宵赶图修正 |
|---|---|---|
| `0-39` | 进度 `-1` | 进度 `-2` |
| `40-59` | 无修正 | 无修正 |
| `60-79` | 进度 `+2`；精力消耗 `-1` | 进度 `+3`；精力消耗 `-1` |
| `80-100` | 进度 `+4`；精力消耗 `-2`；压力增量 `-1` | 进度 `+5`；精力消耗 `-2`；压力增量 `-1` |

- 无论 `软件技术` 多高，`爆肝通宵赶图` 的单次压力增量最低仍为 `+8`，单次精力消耗最低仍为 `-14`。

### 质量类行动

- 直接增加 `作品质量` 的常驻行动只读取 `方案推敲` 与 `场地调研`。
- `阅读/展览/讲座` 主要负责累计 `创意审美`，不直接增加 `作品质量`。
- `方案推敲` 的质量修正按 `设计水平 60% + 创意审美 40%` 计算。
- `场地调研` 的质量修正按 `设计水平 50% + 创意审美 50%` 计算。
- 加权结果统一按下表修正行动的 `基础作品质量`，最终结果向下取整：

| 加权结果 | 质量修正 |
|---|---|
| `< 40` | `-1` |
| `40-59` | `0` |
| `60-79` | `+1` |
| `80-100` | `+2` |

- 质量类行动的单次最终收益最低为 `+1`，避免出现投入行动却完全不涨质量的情况。

## 状态收益修正

重复选择同一行动不做固定衰减，收益由精力和压力决定。

| 状态 | 条件 | 影响 |
|---|---|---|
| 状态良好 | 精力 >= 60 且压力 <= 50 | 正常收益 |
| 状态波动 | 精力 35-59 或压力 51-75 | 正向收益 -10% |
| 状态危险 | 精力 20-34 或压力 76-85 | 正向收益 -20%，给提醒 |
| 崩溃边缘 | 精力 < 20 或压力 > 85 | 正向收益 -35%，事件风险提高 |

提醒只保留三类：

- 精力见底
- 压力过载
- 状态危险

高危行动限制：

| 高危类型 | 条件 | 限制 |
|---|---|---|
| 精力高危 | 精力 < `20` | 禁用高消耗行动：`爆肝通宵赶图 / 接设计外包 / 校外兼职 / 万里路 / 实习开始`；交互事件中高消耗选项置灰 |
| 压力高危 | 压力 > `85` | 禁用高压行动：`爆肝通宵赶图 / 接设计外包 / 高压专属技能 / 高压交互选项`；事件负向权重 +20% |
| 双高危 | 精力 < `20` 且压力 > `85` | 只保留恢复类行动、低风险商店补给和必要结算操作 |

高危提示文案口径：

- 精力高危：`你已经快撑不住了，先恢复精力，否则很多行动无法继续。`
- 压力高危：`压力已经接近失控，继续硬扛会锁住高压行动。`
- 双高危：`当前状态非常危险，本周必须先把自己拉回来。`

## 每周自动恢复

- 每周所有行动、随机事件、交互事件、商店购买、实习收益/代价、自动生活费结算完成后，统一恢复：`精力 +12`、`压力 -10`
- 周恢复不占行动次数，不因 `模型周 / 赶图周 / 考试周` 改变
- 周恢复后再判断是否仍处于高危状态；若仍处于高危，只保留高危限制，不追加额外保底恢复

## 个人属性加成

个人属性会反向影响对应行动收益。

| 属性值 | 加成 |
|---|---|
| 0-39 | 对应收益 -10% |
| 40-59 | 正常 |
| 60-79 | 对应收益 +8% |
| 80-100 | 对应收益 +15% |

属性成长软上限：

- 六项角色属性达到 `80` 后，来自行动、随机事件和商店的正向属性增量减半，向下取整
- 单次原始增量为 `+1` 时，若该属性已经 `>= 80`，本次不再增加
- 同一学期内，同一项角色属性通过行动、随机事件和商店累计最多增加 `8`
- 导师阶段任务、万里路阶段奖励、角色专属技能不受软上限影响

## 随机事件数值结算

事件文本保留在 `events.md`，数值结算统一收口在这里。

收束原则：

- 每条事件单独对应一条数值结算
- 单个事件默认只改 `1-4` 个字段
- 事件默认不直接修改 `GPA / 作品集 / 简历价值 / 竞赛结果`
- 事件导致金钱降到 `0` 时，仍按总规则立即触发 `生活费断裂`
- 所有数值结算后统一做边界截断：`精力 / 压力 / 进度 / 作品质量 / 作品集 / 六项角色属性` 保持在 `0-100`

## 评图

```text
评图总分 = 进度 40% + 作品质量 35% + 汇报策略结果 25% + 导师任务修正
```

等级与绩点：

- A：`4.0`，总分 `>= 85`
- B：`3.3`，总分 `75-84`
- C：`2.7`，总分 `60-74`
- D：`2.0`，总分 `45-59`
- F：`0.0`，总分 `< 45`

规则：

- `D` 视为过关
- `F` 视为挂科
- 每次评图结算完成后，立刻向玩家显示本次评图等级。
- 连续 `2` 次 `F` 直接触发 `连续挂科退学`

## GPA 结算

- 开局显示 `GPA = 4.0`
- 每学期先根据评图等级得到设计课绩点
- 再叠加大学课程修正
- 得到该学期最终绩点后，再并入个人累计 GPA

累计规则：

```text
个人GPA = 所有已完成学期最终绩点的平均值
```

- 锁定 `新媒体内容` 且门槛达标，读取 `内容编辑`；未达标读取 `转行求职`。
- 锁定 `插画师` 且门槛达标，读取 `插画师`；未达标读取 `转行求职`。

结果揭示规则：

- `转行` 路线在大五阶段只完成岗位投递和内部判定
- 是否转行成功统一延后到大五下毕业设计答辩后的结局结算再揭示

## 毕业设计完成门槛

顺利毕业必须完成毕业设计；否则无论路线是否达标，都读取 `没毕业`。

毕业设计完成条件：

- 大五下毕业设计评图等级为 `D` 或以上
- 大五下评图前课题进度 >= `60`
- 大五下评图前作品质量 >= `45`
- 大五下毕业设计答辩一结束，立即进入结局系统结算；此前所有路线考试、申请、投递的内部结果都在此时统一读取

状态收尾：

| 状态收尾 | 条件 |
|---|---|
| 昂扬 | 通关时精力 >= `60` 且压力 <= `50` |
| 复杂 | 通关时不满足昂扬，也不满足沉重 |
| 沉重 | 通关时精力 < `35` 或压力 > `75` |

## 强路线稀有度

强路线不是失败条件，只影响成功通关时的主结局优先级。

| 强度 | 路线 | 目标体感 |
|---|---|---|
| 顶级强路线 | 保研直升、顶级名校留学、选调回乡、大师建筑事务所 | 需要明确规划，普通游玩不稳定达成 |
| 中强路线 | 考研上岸、普通留学、国企设计院、AI 新职业 | 合理规划可达成 |
| 普通路线 | 地方设计院、普通事务所、非 AI 转行、状态毕业 | 大多数未失败玩家可回落 |

调参原则：

- 若强路线过多触发，优先上调 `作品集 / GPA / 实习价值 / AI 相关经历` 门槛
- 若普通通关过难，优先降低失败压力，不降低强路线门槛
- `稳定毕业 / 昂扬毕业 / 带伤毕业` 不读取作品集、竞赛、实习等强路线条件

## 中途失败阈值

- 生活费断裂：任意结算后金钱归零，读取人生结局 `你破产了`
- 压力失控：压力 `> 90` 连续 `2` 周
- 连续挂科退学：连续 `2` 次评图 `F`
- 被迫停学：精力 `= 0` 且压力 `> 90`，并在周末结算时仍未脱离
