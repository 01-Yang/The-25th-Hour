# 个人简历系统旧确认稿

本文不再作为现行规则源。个人简历系统的稳定口径已收敛到 `systems.md` 的“个人简历”章节；路线读取、数值门槛和结局解析仍分别以 `systems.md`、`numbers.md`、`endings.md` 为准。

保留本文只为说明历史来源：此前简历系统已确认“游戏内经历面板”方向，不做真实求职简历生成器，不做简历总分，不提前展示隐藏路线结果。

当前开发只读取以下结论：

- `D` 及以上课程设计成果写入；`F` 不写入。
- 大五毕业设计只在最终答辩后写入一次。
- 竞赛获奖进入简历高亮；未获奖保留在竞赛记录。
- 只有已完成实习进入简历；进行中、退出或作废的实习不展示。
- AI 相关经历作为标签展示。
- 万里路只把完成 12 个地点后的里程碑写入简历。
- 雅思只展示最高分。
- 成长成就、校外兼职流水、普通课程历史和隐藏路线考试结果不写入简历。
