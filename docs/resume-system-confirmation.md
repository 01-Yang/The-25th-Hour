# 个人简历系统确认

个人简历是游戏内的结构化经历面板，不是真实求职简历生成器。路线结算仍读取底层状态，不读取简历展示文案。

## 固定口径

- 不做简历总分。
- 不允许玩家编辑规则经历文案。
- 不弹出“写入简历”提示，写入动作在后台结算。
- 不提前展示路线考试、申请、投递的隐藏结果。
- 不把成长成就、普通课程、校外兼职流水写入简历。

## 写入内容

| 类别 | 写入规则 | 展示位置 |
|---|---|---|
| 基础档案 | 玩家昵称、大学名称、角色、学历背景、家庭背景 | 页头 |
| 核心摘要 | GPA、作品集总分、最高评级、代表课题数量 | 标签摘要 |
| 课程设计成果 | 只有 `B / A / S` 正式评图写入作品集成果；`C / D / F` 不写入作品集；毕业设计只写最终答辩结果 | 主经历流 |
| 竞赛经历 | 获奖进入高亮；未获奖只进竞赛记录 | 主经历流 / 附属记录 |
| 实习经历 | 只写入已完成实习；进行中、退出、作废不展示 | 主经历流 |
| AI 相关经历 | 展示计数与来源标签 | 标签摘要 |
| 建筑生的万里路 | 只写完成 `12` 个地点后的里程碑；地点明细留在旅行记录页 | 主经历流 |
| 雅思成绩 | 只展示最高分和考试年份 | 考试/证书区 |

## 页面结构

- 页头：昵称、大学名称、角色、学历背景、家庭背景、GPA、作品集总分。
- 标签摘要：代表课题、竞赛获奖、实习价值、AI 相关经历、万里路完成度、雅思最高分。
- 主经历流：课程设计成果、竞赛获奖、已完成实习、万里路完成里程碑。
- 附属记录：竞赛未获奖投稿、雅思考试历史。
- 空状态：没有经历时显示轻提示，不弹窗催促。

## 作品集展板图

专业设计作业展板图不进入简历页头，也不作为简历主经历卡面；只在作品集系统的课程设计成果详情页展示。图片路径、`semesterIndex` 映射、评级门槛和大五未挂载口径统一维护在 `portfolio-board-image-map.md`。
