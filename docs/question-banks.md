# 题库总表

本文件统一维护大学课程题库、雅思题库、升学专业题库与行测题库。

## 使用规则

- 本文件只维护题目、选项、答案与题库分组。
- 抽题数量、GPA 修正、考试折算分、展示规则和路线门槛统一维护在 `numbers.md`；流程口径见 `systems.md`。
- 所有答题系统在单题作答后都不显示对错，也不显示正确答案；玩家选择后直接进入下一题。
- 大学课程题用于学期末课程修正；雅思题用于留学申请；升学专业题用于保研复试与考研初试；行测题用于选调、考公和考编。
