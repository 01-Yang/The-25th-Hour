# 结尾回忆走马灯与结局图片匹配稿

本文先作为导演稿和 UI 资产对照表使用，不安装或依赖 HyperFrames 插件。素材来源：

- `结尾回忆走马灯图片/`：107 张 PNG。当前没有发现单独备注文件，本文把图片文件名视为备注。
- `结局图片/`：31 张 PNG。
- `docs/endings.md`：39 条人生结局总库。

## 动画目标

结尾回忆不是普通轮播，而是“毕业照快门按下后，五年记忆被快速翻回”。实现上只使用现有 PNG、覆盖层和字幕，不要求新增分层素材。

成片目标时长：`118s` 左右，允许实现误差 `±8s`。节奏分为 `9` 段：快门触发、高考离家、入学适应、课堂入门、专教赶图、宿舍恋爱、朋友运动、实习现实、毕业告别。

整体画面规格：

- 根画布：`1920x1080`。
- 图片策略：横图全屏 `cover`；竖图或非 16:9 图放进相册卡片，卡片最大宽 `78vw`、最大高 `82vh`。
- 字幕：像素风文字，底部安全区内显示，最多 `2` 行。
- 音乐：只作为背景音乐，不按歌词或鼓点切镜头。

## 已确认播放口径

- 回忆走马灯每局固定播放全量 `107` 张素材。
- 所有玩家看到同一套走马灯，不按玩家实际事件、恋爱、实习、路线或失败状态做个性化筛选。
- 中途失败不播放完整毕业回忆，直接进入对应失败结局。延毕和顺利毕业进入同一套结尾走马灯，再落到对应人生结局；延毕的失败感由最终结局图和结局文案承担，不在走马灯阶段提前断裂。
- 快闪不是随机抽图，而是阶段式照片连发：例如“开学快闪阶段”“专教返工阶段”“宿舍与朋友阶段”“毕业告别阶段”。
- 玩家第一次抵达结局流程时，必须完整播放结尾回忆走马灯。
- 玩家至少完整看过 `1` 次结尾回忆后，后续结局流程允许显示 `跳过回忆` 按钮。
- `跳过回忆` 只跳过结尾回忆走马灯，不跳过最终人生结局、成长成就结算和排行榜写入。
- 点击 `跳过回忆` 后，当前结束曲在 `600ms` 内淡出，直接进入结局系统；如果尚未开始播放结束曲，则不启动结束曲。

## 音乐衔接

- 结束曲抽取、已播放记录、循环播放和字幕规则维护在 `systems.md` 与 `ending-music-and-subtitles.md`。

## 可实现动效组件

所有动效只作用在整张 PNG、图片容器或覆盖层上。不要让 PNG 内的人物、车轮、学士帽、老师手势等内部元素单独运动。

| 动效名 | 实现方式 | 使用场景 | 参数 |
|---|---|---|---|
| `slow_push_in` | 图片 `scale(1.00 -> 1.06)`，容器不动 | 重点回忆、长字幕 | `2.4-5.2s`，`ease: power1.out` |
| `slow_push_out` | 图片 `scale(1.06 -> 1.00)`，透明度不变 | 告别、空镜 | `2.0-3.5s` |
| `card_slide_left` | 图片卡片 `x(90 -> 0)`、`opacity(0 -> 1)` | 中景照片入场 | `0.28s` 入场，停留后 `0.18s` 出场 |
| `card_slide_up` | 图片卡片 `y(80 -> 0)`、`opacity(0 -> 1)` | 宿舍、课堂、记录类照片 | `0.25s` 入场 |
| `flash_cut` | 白色覆盖层 `opacity 0 -> 1 -> 0` | 大段切换、快门感 | 入白 `80ms`，退白 `160ms` |
| `whip_pan` | 容器 `x(-80 -> 0)` 或 `x(80 -> 0)`，加 `filter: blur(8px -> 0)` | 快闪连发 | `0.22s` |
| `redline_overlay` | SVG 红线、圈画、箭头覆盖层淡入 | 评图、返工、实习纠错 | `0.18s` 硬入，停留到切走 |
| `blueprint_line` | 蓝色线条覆盖层 `scaleX(0 -> 1)` | 建院、丁字尺、方案讨论 | `0.35-0.6s` |
| `map_pin_overlay` | 地图 pin 或路线线条覆盖层弹出 | 调研、万里路、骑行 | `scale(0.8 -> 1)`，`0.18s` |
| `darken_hold` | 黑色覆盖层 `opacity 0 -> 0.28` | 焦虑、沉默、现实段 | `0.4s` 淡入，随图结束淡出 |
| `warm_light` | 暖色径向渐变覆盖层 `opacity 0 -> 0.22` | 恋爱、生日、朋友段 | `0.4s` |
| `pixel_caption_pop` | 字幕 `opacity 0 -> 1`、`translateY(8px -> 0)` | 短句快闪 | `0.12s` 入场，随图淡出 |

禁止动效：

- 不做人物走路、回头、挥手、表情变化。
- 不做照片内部物件飞出、独立旋转或单独变形。
- 不做真实 3D 环绕镜头、自动景深、骨骼动画。
- 不做需要识别图片主体位置的效果。
- 不使用复杂 shader、液体扭曲、粒子爆炸或不可控图像变形。

## 转场规则

- 重点图之间使用 `flash_cut` 或 `slow_push_in` 后淡出，不使用花哨转场。
- 快闪段统一使用 `whip_pan`、`card_slide_left`、`flash_cut` 三种，不新增其他转场。
- 压力段使用 `redline_overlay + darken_hold + 3 帧黑场`。
- 温暖段使用 `warm_light + slow_push_in`。
- 告别段使用 `slow_push_out`，画面运动幅度变小。
- 每 `8-14` 张快闪后必须出现 `1` 张中景或重点图，避免全程高速导致观众看不清。

## 结尾交互按钮

结尾动画播放完成后，右下角出现像素风 UI 按钮：`落幕`。按钮不提前出现，不打断最后一张 `离开.png` 的留白；玩家点击后进入最终结局系统或返回结局汇总页，具体跳转由前端流程决定。

`跳过回忆` 按钮只在玩家已经完整看过至少 `1` 次结尾回忆后出现。按钮位置固定在右上角，尺寸小于 `落幕`，不和字幕重叠。点击后不播放剩余走马灯，直接进入结局系统；本次未播放完的结束曲在 `600ms` 内淡出。
