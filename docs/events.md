# 随机事件体系

## 当前口径

- 本文档只维护随机事件体系的结构、事件池、标题、内容和交互选项。
- 事件属性结算、权重公式、阶段奖励和数值平衡统一放在 `numbers.md`。
- 系统规则、触发流程和开放节点统一放在 `systems.md`。
- 事件结构化字段是开发配置，不在本文档逐条维护；本文档保持为事件文案与选项清单。
- 标题和内容以已经拍板的版本为准；已确认文本不再为了凑数量、压缩字数或统一风格而改写。

## 单局体感

- 单局目标事件触发次数：`90-95`，标准目标按 `92` 次校准。
- `92` 次事件发生在 `60` 个正式周里，包含普通随机事件、交互事件、模型周事件和万里路事件。
- 开学典礼、军训、建院生活开始提示、毕业典礼属于固定流程事件，不计入这 `92` 次事件。
- 暑假写生节点单独结算，不计入这 `92` 次事件。
- 大一、大二为高触发期，普通随机事件与交互事件的平均触发概率高于后续学年。
- 一般节奏为：普通随机事件和交互事件通常成对出现；小概率出现连续 `2` 次都是普通随机事件。
- 普通正式周最多触发 `2` 次普通随机事件、交互事件或模型周事件；万里路事件由玩家主动前往地点后必定触发，不计入这个上限。大一、大二启用周保底；大三后连续 `2` 周没有任何事件时，下一周周末保底触发 `1` 次普通随机事件或交互事件。
- 事件库数量可以超过早期目标数；超过后不删确认内容，只调整事件池规模口径。

## 单局事件次数统计

### 按学年统计

| 学年 | 目标触发次数 | 周均体感 | 说明 |
|---|---:|---:|---|
| 大一 | `22` | 约 `1.8` 次/周 | 入学适应期，社交、朋友、学业、暧昧、娱乐类事件权重上调。 |
| 大二 | `22` | 约 `1.8` 次/周 | 专业生活展开期，学业交互、社交交互和 AI 初期事件共同上调。 |
| 大三 | `18` | 约 `1.5` 次/周 | 中段稳定期，维持课程压力、社交和能力成长的混合节奏。 |
| 大四 | `16` | 约 `1.3` 次/周 | 路线选择期，减少纯日常刷屏，保留关键交互和路线相关事件。 |
| 大五 | `14` | 约 `1.1` 次/周 | 毕业设计期，事件密度略降，避免打断毕业设计与结局节奏。 |
| 合计 | `92` | - | 允许单局自然波动到 `90-95` 次。 |

### 按事件池统计

| 事件池 | 单局目标次数 | 说明 |
|---|---:|---|
| 普通随机事件 | `46` | 默认日常池，正负向体感接近 `1:1`。 |
| 交互事件 | `34` | 带 `2` 个显式选项；大一、大二阶段权重更高。 |
| 模型周事件 | `6` | 模型周高权重触发，正负向固定 `1:1`。 |
| 万里路事件 | `6` | 跟随“建筑生的万里路”固定顺序触发；可因玩家主动安排而突破普通周事件上限。 |
| 合计 | `92` | 不含固定流程事件与暑假写生节点。 |

### 固定必触发与保底事件统计

| 类型 | 事件 | 次数口径 |
|---|---|---|
| 固定流程 | 开学典礼 / 军训 / 建院生活开始提示 / 毕业典礼 | 每局固定触发，不计入 `92` 次事件。 |
| 阶段必触发 | 轻松绷住 | 大一下固定触发 `1` 次，计入普通随机事件目标次数。 |
| 保底触发 | 课桌纸条 | 每局至少 `1` 次，计入普通随机事件目标次数。 |
| 阶段保底 | AI 相关事件 | 大一下至大三下总共保底 `4` 次，计入对应事件池目标次数。 |

## 当前事件池规模

- 普通随机事件：首版下限 `80`，当前已收录 `126`
- 交互事件：首版下限 `40`，当前已收录 `50`
- 模型周事件：固定 `12`
- 万里路事件：固定 `12`
- 当前事件库总量：`200`

## 分池职责

- 普通随机事件：直接出现并结算，没有玩家选项。
- 交互事件：出现后给玩家 `2` 个选项。
- 模型周事件：只在模型周高权重触发，正负向固定 `1:1`。
- 万里路事件：跟随“建筑生的万里路”固定顺序触发。
- AI 相关事件不再单独开池，按是否有选择分别放入普通随机事件或交互事件。

## 重复与冷却

- 同标题事件默认 4 周内不重复。
- 少数高贴脸日常事件允许重复，但重复后权重递减。
- 明确允许重复的代表事件：`熬夜冠军`、`老师的法宝`、`依旧看案例`、`CAD又崩溃了`、`已读不回`、`凌晨的陪伴`、`专教日出`
- 明确不作为重复代表事件：`被AI卷醒`
- 所有带选项、带抽奖、带概率分支的事件，统一归入交互事件。
- 普通随机事件抽取时按“负向 / 正向与中性”做体验归一化，保证体感接近 `1:1`，不要求内容表格本身强行等量。
- 开发配置中每条事件必须补 `event_id / pool / phase / sentiment / tags / context_tags / base_weight / cooldown_weeks / repeatable / trigger_rules` 等结构化字段。
- AI 相关事件通过 `ai_related` 和 `ai_experience_delta` 标记，不单独开事件池。

## 触发限定

| 事件 | 限定 |
|---|---|
| 课桌纸条 | 每局至少触发 `1` 次；若大四下结束仍未触发，则在大五上第 `1-4` 周的正向普通随机事件池中保底插入。 |
| 幸运空位 | 期末周专属事件，只允许在每学期第 `5` 周模型周或第 `6` 周赶图周触发；第 `1-4` 周权重为 `0`。 |
| 微信通过 | 带选项，归入交互事件，不放入普通随机事件池。 |
| 轻松绷住 | 大一下必定触发 `1` 次；触发后不再进入随机池。 |
| AI 相关事件 | 大一下至大三下期间总共保底触发 `4` 次；只从已标记 `ai_related` 的普通随机事件和交互事件中抽取，同一 `event_id` 不重复计入保底。 |
| 暧昧与搭讪事件 | 仅在玩家没有女朋友时进入交互事件池；一旦触发 `两情相悦` 成就并记录已有女朋友，后续所有暧昧、搭讪、恋爱推进类事件权重归零。 |

## 固定流程事件

固定流程事件不计入普通随机事件池。开学典礼和军训发生在游戏正式进入周行动前，此时所有行动不可使用；军训结束后进入建院生活提示页，随后选择大一导师，再开始大一上第 `1` 周。

### 开学典礼

开学的新生典礼，你坐在建院报告厅靠走廊的位置，看阳光从高窗斜进来，落在前排女生的马尾辫上。台上的人说着建筑、理想和远方，你一个字都没听进去，只是在想：我会在这里遇见谁？

院长讲话的时候，你掏出手机拍了一张PPT，上面写着“欢迎新同学”。那张照片在你的相册里存了五年，每次清理手机都舍不得删。散场后你站在大学广场上，夕阳把整个校园染成橘色。你打开手机给妈妈发了一条消息：“典礼结束了，一切都好。”她回了一个“好”字。你盯着那个字看了一会儿，觉得这个字比任何建筑都稳。你收起手机，跟着人群往前走，不知道要去哪里，但你知道你会找到的。

| 选项 | 说明 |
|---|---|
| 认真听讲 | 你强打精神记下了几个陌生的词，虽然听不全懂，但至少开始知道自己来到哪里。 |
| 太困了，先睡觉 | 你在掌声里睡了一小会儿，醒来时只记得阳光和那个“欢迎新同学”。 |

### 军训

九月的太阳还毒，你站在操场上，帽檐压得很低，汗水顺着脖子往下淌。教官喊“齐步走”，你同手同脚走了三步，被隔壁的同学小声提醒“换边”。你涨红了脸，偷偷看旁边那个女生，她比你还紧张，步子迈得像在跳格子。休息时你们坐在滚烫的塑胶跑道上，有人递过来一瓶水，你不认识他，他说“都是一个院的”。你拧开盖子，喝了一大口，觉得这大概是大学里第一口甜的。晚上拉歌，你们坐在草坪上，对着星空吼跑了调的《团结就是力量》。你笑到肚子疼，笑完以后抬头看天，忽然觉得，未来好像也没有那么可怕。

| 选项 | 说明 |
|---|---|
| 我是标兵 | 你把每个动作都做到位，晒得发懵，但也开始有点相信自己能撑下来。 |
| 得过且过 | 你混在人群里少晒一点太阳，没出风头，也没有把自己逼太紧。 |

### 建院生活开始提示

军训结束了。你拖着还有点酸的腿走进建院，专教的灯亮着，楼道里有人抱着图板跑过去。你还不知道接下来五年会发生什么，但从这一刻起，你正式成为这里的一部分。

### 毕业典礼一阶段

学士服穿在身上有点大，帽子怎么也戴不正。摄影师喊“三、二、一”，所有人同时把学士帽扔向天空。

帽子落下来的那一秒，你忽然看见了五年前的自己——站在校门口，拖着行李箱，仰头看建院的招牌。

那时候你觉得五年很长，长得像画不完的剖面。现在它短得像一声快门。你站在人群里，左右是熟悉的脸，有些人你甚至叫不全名字，但你知道，你们共享过同一间教室的灯、同一卷硫酸纸的痕迹、同一把美工刀划破手指的疼。

### 毕业典礼二阶段

摄影师说“再来一张”，你把手搭在旁边同学肩上，他冲你比了个耶。你想起某个凌晨，你们一起在专教改图，他分了你半罐红牛，你说“谢了”，他说“别废话，快画”。

那些夜晚回不去了。

快门再次按下，这张照片里，你笑得比第一张自然。

你知道，再过很多年，你翻到这张毕业照时，会记得今天的风、阳光、还有身边这些人的声音。身边的同学在自拍、在拥抱、在大喊“毕业了”，你也跟着笑了。可你知道，你心里有一块地方，安静得像专教深夜的走廊。你问自己：这五年，我到底变成了谁？没有答案。但你知道，你不再是那个会在图纸前哭的小孩了。你对自己说：就这样吧，挺好的。然后你弯腰，捡起掉在地上的学士帽，拍了拍灰，走向下一个路口。

毕业典礼是游戏尾声固定事件，不再结算属性；毕业典礼结束后进入结局系统。
