# 音乐素材清单

本文是音乐素材唯一清单，维护开头曲、五个学年 BGM 和结束曲的实际文件名、时长、播放顺序与结束曲权重。播放流程只在 `systems.md` 定义；结束曲弹窗、字幕和专辑表现只在 `ending-music-and-subtitles.md` 定义。

## 使用口径

- 当前音频文件已放入仓库根目录下的 `中间曲/` 与 `结束曲/`。
- 本文只维护实际文件名、顺序、时长、歌词接入位和结束曲权重。
- 前端播放流程、切歌时机、淡入淡出、跳过回忆和结尾等待态规则回到 `systems.md`。
- 开头曲和中间 BGM 不接歌词文件；结束曲的字幕表现回到 `ending-music-and-subtitles.md`。

## 时长标注规则

- 本文时长已按当前下载的 MP3 文件读取。
- 前端仍建议以音频文件元数据或解码结果为准，本文时长作为内容核对和 UI 标注参考。

```text

```
