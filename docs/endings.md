# 结局与成长成就系统

## 基本规则

结算后统一进入 `结局与成就` 页面，页签固定为：

- 人生结局
- 成长成就
- 排行榜

当前内容量以本文件总库表格为准，不设置固定数量上限。新增或删除条目时，应同步保证每条记录都有 `标题 / 内容 / 触发条件 / 分数`。

结局与成就均计入独立图鉴。人生结局首次解锁时按表格计分；同一人生结局第二次及以后重复达成时，只计入 `50` 分。成长成就全局只计分一次，单存档内也不重复弹窗；之后再次满足条件不再弹窗，也不再重复计分。

## 成就弹窗

成长成就在游戏过程中即时触发。

- 达成条件后立刻弹出成就提示。
- 弹窗显示：成就标题、成就内容、获得分数。
- 弹窗不打断结算、评图、路线选择等关键流程；若正在关键流程中，则延后到该流程结束后依次弹出。
- 成就一经弹出，永久收录进成长成就图鉴；同一成就本存档和历史图鉴内均不重复触发。
- 涉及连续周、任意三周窗口或跨周累计次数的成就，在周结算完成后统一判定。
- 涉及历史游戏的成就读取全局图鉴记录，不只读取当前单局。

## 成就图鉴 UI

- 成长成就图鉴显示全部成就条目。
- 未获得成就显示为灰色锁定态。
- 未获得成就不展示达成条件；内容区域显示锁定占位文案。
- 已获得成就展示标题、内容、首次获得时间和分数。

## 分数规则

最高单项分数为 `1000`。

- 结局分数区间：`100-1000`
- 成就分数区间：`10-800`
- 强路线结局接近高分，普通通关结局保留中低分，失败结局保留少量收集分。

排行榜采用累计口径：

- 排行榜分数 = 已解锁结局分 + 已解锁成就分 + 重复达成人生结局折算分。
- 失败局也可以产生人生结局和成长成就，因此失败局结算后的本地累计总分允许提交排行榜。
- 排行榜只提交同一 `playerId` 的本地历史最高累计总分；重复达成人生结局的 `50` 分允许低幅累计，后续若失衡再设置软上限。

同一成长成就第二次及以后重复达成时不计入排行榜。

## 量化口径

- 高危状态：`精力 < 30` 或 `压力 > 80`。
- 精力危险线：`精力 < 30`。
- 安全区：`精力 >= 35` 且 `压力 <= 75`。
- 金钱高危：当前金钱低于角色家庭背景对应的 `1` 周自动花费。
- 普通：通关时不满足沉重。
- 沉重：通关时 `精力 < 35` 或 `压力 > 75`。
- 正式参与路线：大一至大四不提供后期路线入口，也不记录任何后期路线状态；大五上第 `1` 周起所有后期路线正式参与入口全部开放展示，但不弹出提示弹窗，不显示红点或催促 UI；只有大五阶段完成对应方向的报名、申请、投递、考试或确认后，才形成路线正式参与记录。只满足数值门槛不算正式参与。
- 完成毕业设计：大五上和大五下共同承载同一个毕业设计；大五毕业设计进度与作品质量上限均为 `200`；大五上期中汇报前毕业设计进度必须达到 `100`，否则本次评图为 `F`；大五上作品质量折算后至少达到不被评为 `F` 的门槛；大五下毕业设计最终进度必须达到 `200`，最终作品质量折算后的评图等级为 `D` 及以上。
- 实习价值：按累计分计算。每完成 `1` 段普通实习记 `+1`，每完成 `1` 段强事务所实习记 `+2`，每完成 `1` 段名企实习记 `+3`；退出作废的实习不计入。

## 结局读取规则

路线成功类结局必须同时满足：

- 玩家完整读完五年，且未触发中途失败。
- 玩家已形成且只形成 `1` 个路线正式参与记录。
- 玩家完成毕业设计；未完成时进入内部 `没毕业` 状态，并读取人生结局 `毕业失败——延毕`，不读取成功结局。
- 角色数值、作品集、GPA、竞赛、实习、AI 经历等满足该结局触发条件。
- 需要考试或复试的路线，必须同时满足数值门槛和对应考试分数线。
- 路线失败原因、缺失成功门槛、概率和内部考试线不向玩家精确展示，只通过结局文本表达结果；调试日志内部保留原因。

如果玩家满足某条路线的数值门槛，但没有在过程中选择该路线，不读取该路线结局。

若玩家毕业前没有正式参与任何后期方向，结局系统跳过路线成功与路线回落结局，只按最终状态读取普通状态毕业结局。

路线相关成就只在报名、申请、投递、考试或确认等正式参与动作完成时触发。

路线互斥规则：

- `保研 / 考研 / 留学 / 选调 / 考公 / 考编 / 建筑工作 / 转行` 每局只能选择一个。
- `选调 / 考公 / 考编` 不能互相串线读取结局；正式参与时还必须锁定 `1` 个具体岗位或岗位层级，并且只能读取该岗位成功结局或对应回落结局。
- `建筑工作` 必须锁定 `1` 个投递岗位，并且只能读取该岗位成功结局或 `求职待定——回去等通知吧`。
- `转行` 必须锁定具体岗位，并且只能读取该岗位成功结局或 `转行失败——转行可不是那么容易的！`。
- `创业` 是 `转行` 下的特殊岗位入口，只有六项角色属性全部 `>= 90` 时可选；满足后弹出 `恶魔的契约` 二选一弹窗，签下契约才形成路线正式参与记录并读取 `没有退路的选择`；选择放弃等同于取消本次选择，不锁定路线。

创业契约弹窗：

| 标题 | 内容 | 选项 |
|---|---|---|
| 恶魔的契约 | 现在有一份恶魔契约摆在你的面前，如果签下，你可以拥有无限的心力与精力；但代价是，无论多痛苦你也无法停下脚步。你愿意吗？ | `如果成长的代价是失去，那我无所谓` / `我放弃了` |

失败状态名与结局标题对接：

- 系统内部的 `生活费断裂` 状态读取人生结局 `你破产了——有感觉吗`。
- 系统内部的 `没毕业` 状态读取人生结局 `毕业失败——延毕`。
- 中途失败结局可以包含后日谈；`被迫停学——好好休息，同学` 保留“后来毕业”的尾声，但规则上仍视为本局中途失败并结束。

同一路线内多个结局同时满足时，读取分数最高的结局；若分数相同，读取表格顺序更靠前的结局。所有结局都作为独立图鉴项收集。
