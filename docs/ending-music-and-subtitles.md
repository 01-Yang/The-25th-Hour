# 结束曲与歌词字幕

本文维护结束曲表现、字幕同步、歌曲介绍弹窗和专辑视觉规则。音乐系统流程以 `systems.md` 为准；曲目、文件名、时长和播放权重以 `music-asset-manifest.md` 为准。

## 结束曲启动与抽取口径

结束曲在毕业答辩完成后先完成抽取，但不立刻播放。玩家进入结尾等待态时，音乐保持静默；当玩家按下毕业典礼 / 结尾动画对应的开始按钮时，结束曲与该段动画同帧开始播放。若毕业典礼和回忆走马灯拆成两个按钮，音乐跟随真正启动当前动画的那个按钮，不在等待页提前播放。
结束曲不需要和毕业典礼、回忆走马灯或歌词节点卡点。

抽取使用玩家级播放记录：

- 每位玩家维护一份已播放结束曲记录。
- 玩家通关时，只从该玩家尚未播放过的结束曲中抽取。
- 已经为该玩家播放过的结束曲，在该玩家再次通关时不再进入抽取池。
- 如果该玩家已播放过全部结束曲，则清空该玩家的已播放记录，并从完整曲池重新抽取。
- 曲目抽中后，在本次毕业典礼和结局流程中保持不变，玩家不可切歌。
- 如果曲目播放完时玩家还没有重新开始或离开结局流程，则当前曲目从头重新播放一次；仍停留时继续按同曲循环。

抽取池、`songId`、字幕文件、时长和播放权重统一维护在 `music-asset-manifest.md`。专辑像素图当前为待生成资产，生成提示词维护在 `pixel-art-asset-prompts.md`。

## 歌曲介绍弹窗

结束曲随按钮点击开始播放的第一帧，屏幕中央偏下弹出软弹窗。弹窗自动渐出，不阻塞毕业典礼。

- 显示内容：歌曲名、主唱、`1` 句弹幕介绍。
- 入场：`opacity 0 -> 1`，`translateY(8px -> 0)`，`240ms`。
- 停留：`3200ms`。
- 退场：`opacity 1 -> 0`，`translateY(0 -> -6px)`，`900ms`。
- 玩家不能切歌；如果关闭音乐，弹窗仍显示一次。
- 同一首结束曲在结局流程内循环重播时，不重复弹出歌曲介绍弹窗。

## 底部歌词字幕

只在结束曲播放时显示歌词字幕。开头曲和中间 BGM 不显示歌词。

- 歌词字幕统一使用像素风文字：高可读像素字体、浅色字、深色 `2px-3px` 像素描边、轻微像素投影。
- 字幕动效只做 `opacity`、`translateY` 和逐句切换，不做手写、描边生长、文字粒子或复杂变形。
- 字幕区域固定在游戏底部安全区内，居中显示。
- 同时最多显示 `2` 行，超出内容按歌词文件时间切分。
- 当前句淡入，上一句淡出，切换时间 `180-260ms`。
- 字幕不遮挡关键按钮；毕业典礼阶段若有选择按钮，字幕自动上移。
- 字幕跟随 `audio.currentTime` 或全局音乐时钟同步。
- 游戏暂停时，歌曲和字幕同步暂停。
- 音乐静音时，字幕继续按播放时间显示；若播放停止，字幕停止。
- 缺少授权歌词文件时，不显示字幕，不用占位文本。

推荐 `.lrc` 格式：

```text
[00:00.00] <授权歌词第一句>
[00:04.20] <授权歌词第二句>
```

推荐解析结果：

```ts
interface LyricLine {
  startMs: number;
  endMs?: number;
  text: string;
}
```

## 专辑旋转效果

结束曲播放时展示像素风专辑图。

- 每首歌对应 `1` 张像素风专辑图，图片由玩家后续生成。
- 图片展示为唱片/专辑封面视觉，可在毕业典礼背景 UI 的音乐角标中出现。
- 播放中持续旋转；暂停时停止在当前角度。
- 默认速度：`12s` 一圈。
- 渐出弹窗和底部歌词不跟随专辑旋转。
- 开启动效减弱时，旋转改为极慢呼吸光或静态封面。

## 开头曲与中间曲

- 开头曲：固定音乐，文件名以 `music-asset-manifest.md` 为准；不显示歌词。
- 中间 BGM：按学年分组，玩家不能手动切歌；按该学年的既定顺序自动顺播，播到组尾后回到第 `1` 首循环；不显示歌词。
- 结束曲：毕业答辩完成后抽取；玩家按下毕业典礼 / 结尾动画对应的开始按钮时播放；玩家不可切换；显示弹窗、专辑旋转和底部歌词。

## 实现验收

- 固定 seed 与同一份玩家已播放记录下，结束曲抽取可复现。
- 毕业答辩完成后抽取结束曲，但在结尾等待页不提前播放。
- 玩家按下毕业典礼 / 结尾动画对应的开始按钮时，已抽取歌曲与对应动画同帧开始播放。
- 抽取时排除该玩家已播放过的结束曲。
- 该玩家播放完全部结束曲后，下一次通关会重置已播放记录并重新从完整曲池抽取。
- 结束曲播放完且玩家仍停留在结局流程时，当前曲目从头循环播放。
- 歌曲介绍弹窗只出现一次，并自动渐出。
- 有授权 `.lrc` 时字幕和音频时间同步；无 `.lrc` 时不显示字幕。
- `.lrc` 只保留正式歌词行；作词、作曲、编曲、制作、来源、版权提示等信息不显示，也不保留在字幕文件中。
- 专辑图缺失时显示统一像素占位，不报错。
- 开头曲和中间曲全程不显示歌词。
