# 结束曲与歌词字幕

本文维护结束曲、字幕同步、歌曲介绍弹窗和专辑视觉规则。音乐系统的流程入口仍以 `systems.md` 为准，歌曲内容池仍可在 `content-plan.md` 汇总。

## 版权口径

结束曲均为商业歌曲。仓库不直接保存完整歌词，不在文档中复制整首歌词。实现时只接入玩家已授权或已购买授权的歌词文件，例如 `.lrc` 或 `.vtt`。

开发期允许先保留歌词导入位和校验来源；未导入授权歌词时，底部字幕不显示，歌曲仍正常播放。

## 歌曲介绍弹窗

结束曲开始播放的第一帧，屏幕中央偏下弹出软弹窗。弹窗自动渐出，不阻塞毕业典礼。

- 显示内容：歌曲名、主唱、1 句原创介绍。
- 入场：`opacity 0 -> 1`，`translateY(8px -> 0)`，`240ms`。
- 停留：`3200ms`。
- 退场：`opacity 1 -> 0`，`translateY(0 -> -6px)`，`900ms`。
- 玩家不能切歌；如果关闭音乐，弹窗仍显示一次。
- 弹窗介绍使用原创文案，不直接使用歌词原句，除非后续取得授权。

## 底部歌词字幕

只在结束曲播放时显示歌词字幕。开头曲和中间 BGM 不显示歌词。

- 字幕区域固定在游戏底部安全区内，居中显示。
- 同时最多显示 `2` 行，超出内容按歌词文件时间切分。
- 当前句淡入，上一句淡出，切换时间 `180-260ms`。
- 字幕不遮挡关键按钮；毕业典礼阶段若有选择按钮，字幕自动上移。
- 字幕跟随 `audio.currentTime` 或全局音乐时钟同步。
- 游戏暂停时，歌曲和字幕同步暂停。
- 音乐静音时，字幕继续按播放时间显示；若播放停止，字幕停止。
- 缺少授权歌词文件时，不显示字幕，不用占位文本。

推荐 `.lrc` 格式：

```text
[00:00.00] <授权歌词第一句>
[00:04.20] <授权歌词第二句>
```

推荐解析结果：

```ts
interface LyricLine {
  startMs: number;
  endMs?: number;
  text: string;
}
```

## 专辑旋转效果

结束曲播放时展示像素风专辑图。

- 每首歌对应 `1` 张像素风专辑图，图片由玩家后续生成。
- 图片展示为唱片/专辑封面视觉，可在毕业典礼背景 UI 的音乐角标中出现。
- 播放中持续旋转；暂停时停止在当前角度。
- 默认速度：`12s` 一圈。
- 渐出弹窗和底部歌词不跟随专辑旋转。
- 开启动效减弱时，旋转改为极慢呼吸光或静态封面。

## 开头曲与中间曲

- 开头曲：固定音乐，资源待玩家上传；不显示歌词。
- 中间 BGM：按学年分组，每学年 `3` 首，默认播放第 `1` 首，玩家可在当前学年 BGM 组内切换；不显示歌词。
- 结束曲：毕业典礼前抽取，毕业典礼第一帧播放；玩家不可切换；显示弹窗、专辑旋转和底部歌词。

## 实现验收

- 固定 seed 下结束曲抽取可复现。
- 毕业典礼第一帧开始播放已抽取歌曲。
- 歌曲介绍弹窗只出现一次，并自动渐出。
- 有授权 `.lrc` 时字幕和音频时间同步；无 `.lrc` 时不显示字幕。
- 专辑图缺失时显示统一像素占位，不报错。
- 开头曲和中间曲全程不显示歌词。
