# 第二十五小时：建筑生模拟器

《第二十五小时》是一款建筑学院人生模拟器。玩家会在五年的建院生活中经历课题、赶图、模型、评图、生活费压力、AI 时代冲击与人生岔路口，最终走向各自不同的职业、情绪与人生状态结局。

当前仓库包含两部分内容：

1. 一个早期周循环原型界面
2. 一套已经过初步讨论确认的产品架构文档

## 当前文档

- [PRD](docs/PRD.md)
- [领域词汇](CONTEXT.md)

## 运行

```bash
npm install
npm run dev
```

打开 `http://localhost:3000`。

## 当前状态

代码仍处于非常早期的原型阶段，产品架构与系统边界正在持续整理中。后续开发应优先以 `docs/PRD.md` 和 `CONTEXT.md` 为准，而不是以当前原型代码为准。
